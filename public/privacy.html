Política de Privacidade

Última atualização: março de 2026

1. Sobre a ferramenta

O Instagram Planner Agency é uma ferramenta interna de apoio estratégico para análise de perfis, concorrência e planejamento de conteúdo para Instagram.

2. Dados utilizados

A ferramenta pode acessar, por meio dos tokens configurados, dados básicos das contas conectadas, como nome, username, seguidores, bio, website e informações públicas dos posts.

3. Uso da IA

Os dados enviados para análise textual podem ser processados pela API da Groq para gerar diagnósticos, sugestões estratégicas, análise de concorrência e planejamentos mensais.

4. Armazenamento

Os dados das contas carregadas permanecem em sessão temporária no servidor. O histórico persistente não é o foco principal desta versão.

5. Compartilhamento

Não há comercialização de dados. O uso é interno, limitado ao ambiente operacional da própria agência.

6. Segurança

É responsabilidade do operador manter seguros os tokens do Instagram, a variável de sessão e as chaves do ambiente.
